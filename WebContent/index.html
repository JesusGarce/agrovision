Sobre AgroVision

AgroVision es una aplicación web gratuita que, a partir de una simple imagen de tu cultivo, es capaz de detectar cuál es la plaga que le está afectando y qué tratamiento usar contra ella.

Para ello, nos apoyamos en la tecnología API Google Vision para el análisis de las imagenes y, con nuestra base de datos, encontramos el tratamiento deseado.

Pero no leas más y... ¡pruébalo!

¡Comienza ya!
